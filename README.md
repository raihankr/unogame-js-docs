# UNOGame.js
A lightweight UNO Game Engine written in JavaScript.

# Installation
* Install with npm
  ```sh
  npm install unogame-js
  ```

# How to use
Visit the [tutorial page](https://raihankr.github.io/unogame-js-docs) for a basic setup on starting a new Uno game.

# API Documentation
